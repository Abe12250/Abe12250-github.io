My Selected Data Set

To complete my assignment, I explored various data sets from the awesome-datascience GitHub repository.

Data Set Name: Data Science - 9 Steps Courses

URL: https://www.coursera.org/specializations/jhu-data-science

Potential Applications and Values: I have enjoyed MOOC-style learning courses which have a well-designed curriculum with self-pacing, video and audio lectures, an interactive learning environment, and student comments. This specialization on Coursera is a comprehensive data set of a learning curriculum that outlines the necessary steps and topics to gain a fundamental understanding of data science. The value lies in its structured format which can be used as a blueprint for learning and teaching data science concepts, and it's from a reputable source, Johns Hopkins University.
